Set GraphQL URL for Apollo Client by adding the following field to `.env` file.
```
GRAPHQL_URL=
```

```
$ yarn
$ yarn dev
```

To use Firebase Auth, you will need to run `now dev` and have a `.env` file similar to this.

```
FIREBASE_API_KEY=
FIREBASE_APP_ID=
FIREBASE_AUTH_DOMAIN=
FIREBASE_PROJECT_ID=
```
